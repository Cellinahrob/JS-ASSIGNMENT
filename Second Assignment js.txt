1 Declaration
Delaration of variable means to tell compiler there is a variable\
function of particular data type. You are declaring that something exists, such as a class, function or variable. 


2 Definition of variable means asking compiler to allocate memory to 
variable or define storage for that variable You define how something is implemented, such as a class, function or variable.

3 Initialization is the means of assigning an initial value to a variable. 
So undefined is the initial value of a declared variable.
